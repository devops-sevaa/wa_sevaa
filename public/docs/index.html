Get Started

We provide open source APIs to get hospital bed status in Banglore region. We will extend our APIs to different cities and add further in the coming weeks to provide useful data to the developer community.

All requests should be directed to sevaprod.sagisu.com (Auth tokens not required)

{{url}} = https://sevaprod.sagisu.com/v1

/bedStatus

GET : {{url}}/bedStatus

Provides bed count for hospitals listed on the website. API sorts in descending order of total beds available in each hospital.

Note : Query param can be used to sort on ICU count or ICU ventilator count

Query Params

sortKey : icu or icuVentilator

sortOrder : 1(ascending) or -1(descending)

Sample request : https://sevaprod.sagisu.com/v1/bedStatus

Sample response : 

              
                {
                   "data": [
                      {
                          "_id": "6092345687e565515850cb7d",
                          "name": "ESIMC PGIMSR MODEL HOSPITAL",
                          "address": " 620/A/1, 12th Main Rd, 2nd Block, Rajajinagar, Bengaluru, Karnataka 560010, India",
                          "mobile": " 918023325130",
                          "general": 11,
                          "hdu": 0,
                          "icu": 0,
                          "icuVentilator": 0,
                          "total": 11
                      }
                   ]
                }

Response fields

Key Description
_id Unique ID of the hospital
name Name of the hospital
address Address of the hospital
mobile Mobile number of the hospital
general Number of beds available in general ward
hdu Number of beds available in HDU ward
icu Number of beds available in ICU
icuVentilator Number of beds available in ICU ventilator
total Total number of beds available in the hospital across all the wards

POST : {{url}}/reserve

API to make hospital bed bookings.

Sample request : https://sevaprod.sagisu.com/v1/reserve

Body params : json of booking information 

              
                 
                {
                  "reserve":{
                     "type":"ICU"
                  },
                  "patientName":"Test User",
                  "age":"20",
                  "location":"User location",
                  "onOxygenCylinder":false,
                  "hospitalsVisited":[A,B,C],
                  "testedForCovid":true,
                  "covidResult":"Negative",
                  "hospitalPreference":"Private",
                  "attenderMobile":"XXXXXXXXXX",
                  "relationship":"wife",
                  "srfid":"900",
                  "buNumber" : "900",
                  "user":"60b90516499210371968513f"
               }

Request fields

Key Validation Description
reserve.type required,string Type of the bed requested
patientName required,string Name of the patient
age required,string Age of the patient
location required,string Address of the patient
onOxygenCylinder required,boolean Is pateint on oxygen cylinder or not
hospitalsVisited string array List of all the hospitals visited before booking here
testedForCovid required,boolean Is covid test is done or not?
covidResult required,string Covid report details. POSITIVE,NEGATIVE,AWAITING are the values to this field
hospitalPreference required,string Preferences for the hospital. GOVERNMENT,PRIVATE or ANY should be the values to this field
attenderMobile required,string Mobile number of the attender to which the booking details will be sent.
relationship required,string Relaionship of the attender with the patient
srfid string SRFID given during covid test
buNumber string BU Number given during covid test
user string ID of the user, you will get it during login
